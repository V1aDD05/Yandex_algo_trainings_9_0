# Закраска клеток

[Задача 8.1 в Тренировках по алгоритмам 9.0](https://new.contest.yandex.ru/contests/89540/problems?id=30404%2F2026_03_01%2FZSSos64DZz)

Робот находится на бесконечном клетчатом поле. Программа для робота состоит из команд "U" (сдвинуться на клетку вверх), "R" (сдвинуться на клетку вправо), "D" (сдвинуться на клетку вниз) и "L" (сдвинуться на клетку влево). Робот закрашивает все клетки, на которых он побывал, включая начальную.

Определите, сколько клеток будут закрашены более одного раза.

## Примечание по реализации
Симуляция выполняется непосредственно в main процессе чтения данных.

Это позволяет упростить код для такой логики (простой, линейной).

## Формат ввода

В единственной строке содержится строка с программой для робота, она состоит только из символов "U", "R", "D" и "L" и ее длина не превосходит 1000.

## Формат вывода

Выведите количество клеток, закрашенных более одного раза.

### Пример 1

Ввод

```
ULDR
```

Вывод

```
1
```

### Пример 2

Ввод

```
ULRD
```

Вывод

```
2
```
