# Не в своей тарелке

На обед в китайский ресторан пришли `n` гостей, занумерованных числами от `1` до `n`, и сели за круглый стол. Изначально шеф-повар подготовил для `i`-го гостя блюдо с номером `ai`​, при этом все блюда различны и занумерованы числами от `1` до `n`.

У `i`-го гостя аллергия на блюдо с номером `i`. К счастью, стол в китайском ресторане круглый и вращающийся (этот стол вращается только в одну сторону). За одну операцию можно повернуть его так, что блюдо, стоявшее у гостя с номером `n` окажется у гостя с номером `1`. У всех остальных гостей, кроме первого, окажется блюдо, ранее стоявшее у гостя с номером на один меньше.

Определите минимальное количество операций необходимое для того, чтобы ни один гость не получил блюда, на которое у него аллергия.

## Формат ввода

В первой строке задано целое число `n` — количество гостей и блюд (`1≤n≤2⋅10^5`).

Во второй строке задано `n` различных целых чисел `a1​,a2​,…,an`​, где `ai`​ — номер блюда, которое приготовил шеф-повар для `i`-го гостя (`1≤ai​≤n`, если `i!=j`, то `ai​!=aj`​).

## Формат вывода

Выведите единственное число `k` — минимальное число операций поворота стола. Если невозможно повернуть стол так, чтобы ни у одного гостя не было аллергии на его блюдо, выведите число `-1`.

### Пример 1

Ввод

```
5
1 4 2 3 5
```

Вывод

```
2
```

### Пример 2

Ввод

```
5
1 3 5 2 4
```

Вывод

```
-1
```
