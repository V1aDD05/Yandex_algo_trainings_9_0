# Большое и кратное трем

[Задача 6.1 в Тренировках по алгоритмам 9.0](https://new.contest.yandex.ru/contests/89538/problems?id=30404%2F2026_03_01%2FGcSoXQkF3S)

Нужно из набора цифр составить наибольшее возможное число, которое будет делится на три без остатка. При этом, число может начинаться с ведущих нулей, и при равных значениях большим считается более длинное. Например, "00021" считается большим, чем "021".

## Формат ввода

Вводится строка, состоящая из цифр от `0` до `9` — набор цифр. Длина строки не меньше трех и не превосходит `10^5`.

## Формат вывода

Выведите наибольшее число, которое можно составить из данных цифр, чтобы оно делилось на три без остатка. Гарантируется, что это возможно.

### Пример 1

Ввод

```
105
```

Вывод

```
510
```

### Пример 2

Ввод

```
2222
```

Вывод

```
222
```

### Пример 3

Ввод

```
000
```

Вывод

```
000
```

### Пример 4

Ввод

```
54321
```

Вывод

```
54321
```
