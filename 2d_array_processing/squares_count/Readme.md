# Подсчет квадратов

[Задача 10.1 в Тренировках по алгоритмам 9.0](https://new.contest.yandex.ru/contests/89542/problems?id=30404%2F2026_03_01%2F4ZbscdEIOI)

На снимок, сделанный электронным микроскопом, попали квадратные бактерии. Необходимо подсчитать число бактерий на снимке. Гарантируется, что все бактерии имеют форму квадрата со сторонами, параллельными краям снимка.

## Формат ввода

В первой строке вводятся числа `n` и `m` — размеры снимка (`1≤n,m≤500`).

В следующих `n` строках вводится по `m` символов. Каждый символ может быть "#" — часть бактерии или "." — пустое место.

## Формат вывода
Выведите количество бактерий на снимке.


### Пример 1

Ввод

```
8 6
......
...##.
...##.
......
.###..
.###..
.###..
......
```

Вывод

```
2
```

### Пример 2

Ввод

```
2 1
.
.
```

Вывод

```
0
```
