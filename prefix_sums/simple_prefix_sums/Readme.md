По данной последовательности `a1​,a2​,…,an`​ вычислите последовательность ее префиксных сумм `b1​,b2​,…,bn`

## Примечание по реализации
Вычисление префиксных сумм выполняется непосредственно в main процессе чтения данных.

Это позволяет:
- избежать хранения всей последовательности чисел;
- упростить код для такой логики (простой, линейной).

При необходимости вынести логику в отдельную функцию (например, для юнит-тестов), логика легко выносится в функцию, принимающую `const std::vector<int>&`​

## Формат ввода

В первой строке дано целое число `n` (1≤n≤10^3)  — количество элементов в последовательности `a`. Во второй строке дано `n` целых чисел `a1​,a2​,…,an`​ (∣ai​∣≤10^6)  — элементы последовательности.

## Формат вывода

Выведите `n` целых чисел `b1​,b2​,…,bn`​  — последовательность префиксных сумм для последовательности `a`.

### Пример

Ввод

```
5
10 -4 5 0 2
```

Вывод

```
10 6 11 11 13 
```
