# Сумма на отрезке без изменений

[Задача 3.B в Тренировках по алгоритмам 9.0](https://new.contest.yandex.ru/contests/89523/problems?id=928%2F2026_02_18%2F6ruPHNt9QT)

Дан целочисленный массив `A`, состоящий из `N` элементов.

Обработайте `Q` запросов следующего вида:

- `L` `R` — определить сумму элементов с индексами от `L` до `R` включительно.
    

Так как количество запросов в этой задаче велико, их аргументы не будут описаны во входных данных. Вместо этого вам нужно получить их при помощи генератора случайных чисел.

Вам нужно сгенерировать последовательность чисел `X0`​, `X1`​, ..., `X2Q−1`​. Число `X0`​ указано **во входных данных**, а остальные числа вычисляются по формуле `Xi​=(11173⋅Xi−1​+1)mod1000000007`.

Далее из этих чисел образуются аргументы запросов:

- `L0​=min(X0​modN,X1​modN)`, `R0​=max(X0​modN,X1​modN)`;
    
- `L1​=min(X2​modN,X3​modN)`, `R1​=max(X2​modN,X3​modN)`;
    
- ...
    
- `LQ−1​=min(X2Q−2​modN,X2Q−1​modN)`, `RQ−1​=max(X2Q−2​modN,X2Q−1​modN)`.
    

Для большей ясности прочитайте примечание к примеру.

## Формат ввода

Первая строка содержит целое число `N` (1≤N≤10^5) — размер массива.

Вторая строка содержит `N` целых чисел `Ai`​ (0≤Ai​≤10^9) — элементы массива.

Третья строка содержит целое число `Q` (1≤Q≤10000) — количество запросов.

Четвёртая строка содержит целое число `X0`​ (0≤X0​≤10^9) — начальное значение генератора случайных чисел.

## Формат вывода

Выведите одно целое число — остаток от деления суммы ответов на все запросы на `1000000007`.

### Пример 1

Ввод

```
5
1 2 3 4 5
3
123
```

Вывод

```
27
```

### Пример 2

Ввод

```
7
34 10 49 32 28 15 70
5
0
```

Вывод

```
492
```

## Примечание

Последовательность `X` в первом примере равна { 123 , 1374280 , 354830336 , 519316381 , 321884300 , 413258729 } {123,1374280,354830336,519316381,321884300,413258729} Запросы:

- `L0`​=0, `R0`​=3 (ответ 10);
    
- `L1`​=1, `R1`​=1 (ответ 2);
    
- `L2`​=0, `R2`​=4 (ответ 15).
    

Сумма ответов на все запросы равна `27`.
