В школу бальных танцев записались `n` учеников — мальчиков и девочек. Балетмейстер построил их в один ряд, и хочет выбрать из них для первого занятия группу стоящих подряд учеников (менять местами учеников нельзя), в которой количество мальчиков и девочек одинаково. Сколько вариантов выбора есть у балетмейстера?

## Формат ввода

В первой строке задано число `n` (`1≤n≤10^6`). Во второй строке задается описание построенного ряда из мальчиков и девочек — строка из `n` символов `a` и `b` (символ `a` соответствует девочке, а символ `b` — мальчику).

## Формат вывода

В единственной строке должно содержаться единственное число — количество вариантов выбора требуемой группы.

### Пример

Ввод

```
8
aabbaabb
```

Вывод

```
10
```
