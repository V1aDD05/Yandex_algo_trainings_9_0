В постфиксной записи (или обратной польской записи) операция записывается после двух операндов. Например, сумма двух чисел `A` и `B` записывается как `A B +`. Запись `B C + D *` обозначает привычное нам `(B + C) * D`, а запись `A B C + D * +` означает `A + (B + C) * D`. Достоинство постфиксной записи в том, что она не требует скобок и дополнительных соглашений о приоритете операторов для своего чтения.

## Примечание по реализации
Вычисление выражения в постфиксной записи реализовано непосредственно в main процессе чтения данных. Основная причина - обработка в данном случае сильно связана со считыванием данных.

Такой подход позволяет:
- избежать хранения всех операций и операндов;
- упростить код для такой логики (простой, линейной).


## Формат ввода

В единственной строке записано выражение в постфиксной записи, содержащее цифры и операции `+`, `-`, `*`. Цифры и операции разделяются пробелами. В конце строки может быть произвольное количество пробелов.

## Формат вывода

Необходимо вывести значение записанного выражения.

### Пример

Ввод

```
8 9 + 1 7 - *
```

Вывод

```
-102
```
