# Yandex_algo_trainings_9_0
Solutions of training and opetition tasks from Yandex Algorithm trainings 9.0 (March-April 2026). Repository includes only solved tasks.

Репозиторий содержит решения задач, представленных на мероприятии "Тренировки по алгоритмам 9.0". В одной из папок будут публиковаться решения соревновательных задач (тех, что получилось решить), в другой - тренировочных.
