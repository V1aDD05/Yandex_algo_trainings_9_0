В ряд выложено `n` конфет. У каждой конфеты есть тип `pi`​. Можно выбрать последовательность подряд идущих конфет с одним условием — в этой последовательности должно быть ровно два различных типа конфет.

Определите максимальное количество конфет, которое можно взять.

## Формат ввода

В первой строке вводится одно натуральное число `n` (`n≤10^6`) — количество конфет.

Во второй строке вводится `n` целых чисел `pi`​ (`1≤pi​≤10^9`), где `pi`​ — тип `i`-й конфеты.

## Формат вывода

Выведите максимальное количество конфет, которое можно взять.

### Пример 1

Ввод

```
6
3 3 1 2 2 1
```

Вывод

```
4
```

### Пример 2

Ввод

```
2
1 1
```

Вывод

```
0
```
