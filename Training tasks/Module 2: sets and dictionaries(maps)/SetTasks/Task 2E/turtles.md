Широко известна следующая задача для младших школьников. Три черепахи ползут по дороге. Одна черепаха говорит: “Впереди меня две черепахи”. Другая черепаха говорит: “Позади меня две черепахи”. Третья черепаха говорит: “Впереди меня две черепахи и позади меня две черепахи”. Как такое может быть? Ответ: третья черепаха врет! По дороге одна за другой движутся N черепах. Каждая черепаха говорит фразу вида: “Впереди меня a i ai​ черепах, а позади меня b i bi​ черепах”. Ваша задача определить, сколько самое большее количество черепах могут говорить правду.

## Формат ввода

В первой строке вводится целое число N (1 ≤ N ≤ 10000) строк, содержащих целые числа a i ai​ и b i bi​, по модулю не превосходящие 10000, описывающие высказывание i-ой черепахи.

## Формат вывода

Выведите целое число M – максимальное количество черепах, которые могут говорить правду.

### Пример 1

Ввод

```
3
2 0
0 2
2 2
```

Вывод

```
2
```

### Пример 2

Ввод

```
5
0 4
1 3
2 2
3 1
4 0
```

Вывод

```
5
```
