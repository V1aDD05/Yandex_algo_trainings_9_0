# Поиски ровера

Ровер-доставщик выехал из точки `0` в начале проспекта (проспект это прямая). В какой-то момент спутниковая навигация отключилась и точное положение ровера (точка `X`) оказалась неизвестной.

По сигналу с `n` базовых станций мобильной сети удалось определить `n` пар чисел: `xi`​ `di`​, означающих, что базовая станция в точке `xi`​ находится от положения ровера `X` на расстоянии не более чем `di`​.

Определите наибольшее возможное значение `X`.

## Формат ввода

В первой строке вводится одно натуральное число `n` (`1≤n≤10^5`).

В следующих `n` строках вводятся по два целых числа `xi`​ `di`​ (`0≤xi​≤10^9`, `0≤di​≤10^9`).

## Формат вывода

В единственной строке выведите наибольшее `X`, удовлетворяющее всем показаниям сигналов с базовых станций, либо `-1`, если не существует `X`, удовлетворяющего всем показаниям.

### Пример 1

Ввод

```
2
1 2
3 2
```

Вывод

```
3
```

### Пример 2

Ввод

```
3
2 2
6 2
8 1
```

Вывод

```
-1
```

### Пример 3

Ввод

```
5
100 97
115 104
97 115
111 115
107 97
```

Вывод

```
197
```
