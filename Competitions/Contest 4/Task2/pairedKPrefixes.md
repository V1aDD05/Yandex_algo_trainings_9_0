# Парные k-префиксы

|   |   |   |   |
|---|---|---|---|
|Язык|Версия|Время|Память|
|Все языки|—|2 с|256.0 Мб|

|   |   |
|---|---|
|Ввод|стандартный ввод или input.txt|

|   |   |
|---|---|
|Вывод|стандартный вывод или output.txt|

Есть `n` слов одинаковой длины. Необходимо найти такое наибольшее число `k`, чтобы можно было разбить слова на пары так, чтобы у каждой пары слов совпадало хотя бы `k` первых букв.

Определите максимальное значение `k`.

## Формат ввода

В первой строке вводится целое число `n` — количество слов (1⩽n⩽2⋅10^5, `n` — четное).

В следующих `n` строках заданы слова. Гарантируется, что все слова имеют одинаковую длину и суммарная длина слов не превышает 2 ⋅ 1 06 2⋅106.

## Формат вывода

В единственной строке выведите число k — искомое максимальное значение.

### Пример 1

Ввод

```
4
bbbb
aabc
aacc
bbbd
```

Вывод

```
2
```

### Пример 2

Ввод

```
2
a
b
```

Вывод

```
0
```
