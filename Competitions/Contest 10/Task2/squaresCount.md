# Подсчет квадратов

|   |   |   |   |
|---|---|---|---|
|Язык|Версия|Время|Память|
|Все языки|—|1 с|256.0 Мб|

|   |   |
|---|---|
|Ввод|стандартный ввод или input.txt|

|   |   |
|---|---|
|Вывод|стандартный вывод или output.txt|

На снимок, сделанный электронным микроскопом, попали квадратные бактерии. Необходимо подсчитать число бактерий на снимке. Гарантируется, что все бактерии имеют форму квадрата со сторонами, параллельными краям снимка.

## Формат ввода

В первой строке вводятся числа n n и m m — размеры снимка ( 1 ≤ n , m ≤ 500 1≤n,m≤500).

В следующих n n строках вводится по m m символов. Каждый символ может быть "#" — часть бактерии или "." — пустое место.

## Формат вывода
Выведите количество бактерий на снимке.


### Пример 1

Ввод

```
8 6
......
...##.
...##.
......
.###..
.###..
.###..
......
```

Вывод

```
2
```

### Пример 2

Ввод

```
2 1
.
.
```

Вывод

```
0
```
