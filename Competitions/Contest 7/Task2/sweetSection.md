В ряд выложено n n конфет. У каждой конфеты есть тип p i pi​. Можно выбрать последовательность подряд идущих конфет с одним условием — в этой последовательности должно быть ровно два различных типа конфет.

Определите максимальное количество конфет, которое можно взять.

## Формат ввода

В первой строке вводится одно натуральное число n n ( n ≤ 1 06 n≤106) — количество конфет.

Во второй строке вводится n n целых чисел p i pi​ ( 1 ≤ p i ≤ 1 09 1≤pi​≤109), где p i pi​ — тип i i-й конфеты.

## Формат вывода

Выведите максимальное количество конфет, которое можно взять.

### Пример 1

Ввод

```
6
3 3 1 2 2 1
```

Вывод

```
4
```

### Пример 2

Ввод

```
2
1 1
```

Вывод

```
0
```
