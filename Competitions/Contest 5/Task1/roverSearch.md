# Поиски ровера

|   |   |   |   |

|---|---|
|Ввод|стандартный ввод или input.txt|

|   |   |

|Вывод|стандартный вывод или output.txt|

Ровер-доставщик выехал из точки 0 в начале проспекта (проспект это прямая). В какой-то момент спутниковая навигация отключилась и точное положение ровера (точка X X) оказалась неизвестной.

По сигналу с n n базовых станций мобильной сети удалось определить n n пар чисел: x i xi​ d i di​, означающих, что базовая станция в точке x i xi​ находится от положения ровера X X на расстоянии не более чем d i di​.

Определите наибольшее возможное значение X X.

## Формат ввода

В первой строке вводится одно натуральное число n n ( 1 ≤ n ≤ 1 05 1≤n≤105).

В следующих n n строках вводятся по два целых числа x i xi​ d i di​ ( 0 ≤ x i ≤ 1 09 0≤xi​≤109, 0 ≤ d i ≤ 1 09 0≤di​≤109).

## Формат вывода

В единственной строке выведите наибольшее X X, удовлетворяющее всем показаниям сигналов с базовых станций, либо -1, если не существует X X, удовлетворяющего всем показаниям.

### Пример 1

Ввод

```
2
1 2
3 2
```

Вывод

```
3
```

### Пример 2

Ввод

```
3
2 2
6 2
8 1
```

Вывод

```
-1
```

### Пример 3

Ввод

```
5
100 97
115 104
97 115
111 115
107 97
```

Вывод

```
197
```
